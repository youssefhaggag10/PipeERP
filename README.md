# PipeERP

برنامج ERP محلي لإدارة مصانع المواسير البلاستيك، يعمل Offline على جهاز ويندوز أو شبكة داخلية صغيرة.

## الهدف

بناء منتج قابل للبيع لأكثر من مصنع، وليس برنامجًا خاصًا بمصنع واحد فقط.

## التقنية

- Python 3.11+
- PySide6
- SQLite في الإصدار المحلي
- واجهة عربية RTL
- قابل للتطوير لاحقًا إلى PostgreSQL أو نسخة شبكية

## الوظائف المنفذة حاليًا

- تشغيل تطبيق Desktop عربي
- تسجيل الدخول وتشفير كلمات المرور
- إدارة الأصناف والخامات والمنتجات والهالك وقطع الغيار
- إدارة الموردين والعملاء
- مخزن المصنع وتسويات المخزون
- أوامر الشراء والاستلام مع أرقام الدفعات
- أوامر البيع والتسليم مع منع الرصيد السالب
- كارت الصنف وربط الحركات بالمورد أو العميل
- مؤشرات أولية للمخزون
- طبقة Database وRepositories منفصلة
- نظام Database Migrations بإصدارات متتابعة

## الوظائف القادمة

- تقييم المخزون وتكلفة الصرف بطريقة صحيحة
- تتبع Lots عند البيع والتصنيع
- أوامر شراء وبيع متعددة البنود
- التصنيع بالوزن
- الهالك وإعادة استخدامه
- التقارير المالية وتقارير التكلفة
- إدارة المستخدمين والصلاحيات

## التشغيل

```bash
python -m venv .venv
source .venv/bin/activate  # Linux / macOS
# أو على Windows:
# .venv\Scripts\activate

pip install -r requirements.txt
```

### أول تشغيل فقط

عند إنشاء قاعدة بيانات جديدة، حدد كلمة مرور مدير النظام في متغير البيئة `PIPEERP_ADMIN_PASSWORD` قبل التشغيل.

Linux / macOS:

```bash
export PIPEERP_ADMIN_PASSWORD="choose-a-strong-password"
python main.py
```

Windows PowerShell:

```powershell
$env:PIPEERP_ADMIN_PASSWORD="choose-a-strong-password"
python main.py
```

يتم إنشاء المستخدم `admin` فقط عندما تكون قاعدة البيانات بلا مستخدمين ويكون المتغير موجودًا. في التشغيلات التالية لا يلزم إبقاء المتغير مضبوطًا.
